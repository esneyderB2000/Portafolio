Ejercicio 4: periodo

Dado un capital inicial P equivalente a $800.000, se desea encontrar el valor futuro F para las siguientes tasas de interés (i1=0.02, i2=0.08) con períodos (n1= 5, n2=13) respectivamente. Ejercicio 6: almacen

Un almacén de venta de ropa tiene las siguientes promociones para sus clientes, las cuales consisten en lo siguiente (pago en efectivo = E, pago con Tarjeta de Crédito = T): a. Para ventas menores ó iguales a 100.000, con pago en efectivo se hace un descuento del 20% y si es con tarjeta de crédito se hace el 10%. b. Para ventas mayores a 100.000 y menores o iguales a 200.0000, con pago en efectivo se hace un descuento del 30%, con tarjeta de crédito se hace el 15%. c. Para ventas mayores a 200.000, con pago en efectivo se hace un descuento del 40% y con tarjeta de crédito se hace el 20%. Indique el valor del descuento y el total a pagar.

Ejercicio 10: salario
